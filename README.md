# PDF-tool
Tool for modifying PDFs in different ways.

## Features
- Automatically create bookmarks for a PDF
- Create bookmarks for multiple PDF files at a time
- Combine PDF files
- Simple GUI

## Installation
- Install Python 3
- Install requirements:
```
pip install -r requirements.txt
```

## Usage
```
py main.py
```

## Upcoming features
