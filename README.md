# PDF-tool
Tool for modifying PDFs in different ways.

## Features
- Automatically create bookmarks for a PDF

## Installation
- Install Python 3
- Install requirements:
```
pip install -r requirements.txt
```

## Usage
```
py PDF_tool.py
```
or
```
py PDF_tool.py filename.pdf
```
